# Testcase-Generator
競技プログラミングの問題のテストケースを生成します。

main.pyを実行して必要事項を記入するとテストケースが生成されます。
out生成に使うコードは一応python/cpp/javaに対応しているはずですが、cpp/javaはまだ検証できていません。

一つのファイルで複数のテストケースを試すことができます。その場合は「一つのファイルにつき複数のテストケースでテスト」のチェックボックスにチェックしてください。

**一行目**に一つ整数を追加するには、その値の下限、上限を設定します。一行目の数値にのみ、データタイプというものが割り当てられます。

## データタイプについて

None/Integer/Stringの三種類があります。

None: 後のデータ生成にかかわらない数値です。

Integer: 二行目以降で整数列を一列生成し、その長さとして使われます。数列の最小値、最大値、重複を許可するかを追加で設定する必要があります。

String: 二行目以降で長さが固定の文字列からなる列を一列生成し、その際の数列として使われます。文字列の長さ、大文字、小文字のアルファベットを用いることができるか、追加で使用できる文字、重複を許可するかを別に設定する必要があります。

## 使用例
以下のような構造の入力例と出力例を8個ずつ生成するとします。

入力:
```
T
case1
case2
...
caseT
```

$i$ 個目のテストケース:
```
X_i Y_i L_i
A_i1 A_i2 A_i3 ... A_iX_i
S_i1 S_i2 S_i3 ... S_iY_i
```
・ Tは $1$ 以上 $100$ 以下の整数

・ X_iは $1$ 以上 $5000$ 以下の整数

・ A_ijは $1$ 以上 $2026$ 以下の整数

・ Y_iは $1$ 以上 $1000$ 以下の整数

・ L_iは $1$ 以上 $10$ 以下の整数

・ S_ijは $L_i$ 文字の小文字のアルファベット, `#`, `@` のみからなる文字列

・ それぞれのテストケースについて同じ数値は含まれてもよいが、同じ文字列は含まれてはならない。
